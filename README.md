# Cofferer - Coffee Consumption Tracker

A simple Android app to track your daily coffee intake.

## Features

- **Home**: Tap the coffee cup icon to log each coffee you drink.
- **Stats**: View counters for coffees consumed in the last day, week, month, and year. Includes a bar chart for the last 7 days.
- **Profile**: Placeholder for future preferences.

## Getting Started

1. Install dependencies: `npm install`
2. Start the development server: `npx expo start`
3. Run on Android emulator or device: Press `a` in the terminal, or use Expo Go app.

#[.gradle](.gradle)# Building for Android

To build an APK: `npx expo build:android`
